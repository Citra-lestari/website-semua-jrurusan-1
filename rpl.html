Jurusan RPL

APA SIH ITU RPL??

Jurusan Rekayasa Perangkat Lunak (RPL) adalah bidang studi yang mempelajari tentang pengembangan perangkat lunak, mulai dari pembuatan, pemeliharaan, hingga manajemen organisasi dan kualitasnya. RPL menggabungkan prinsip rekayasa dengan ilmu komputer untuk menghasilkan perangkat lunak yang efisien, efektif, dan andal.

Hal-hal yang dipelajari

Bahasa pemrograman

Mahasiswa akan mempelajari berbagai bahasa pemrograman yang umum digunakan dalam pengembangan perangkat lunak, seperti Java, Python, C++, dan lain-lain.

Basis data

Mahasiswa akan mempelajari tentang pengelolaan basis data yang efisien untuk menyimpan dan mengakses data yang dibutuhkan dalam perangkat lunak.

Pengujian perangkat lunak

Mahasiswa akan mempelajari tentang teknik pengujian perangkat lunak untuk memastikan kualitas dan keandalan perangkat lunak yang dihasilkan.

Pengembangan Web

Mempelajari teknologi dan framework untuk pengembangan aplikasi web, baik frontend (tampilan) maupun backend (logika dan server).

Jenjang Kerja

Web Developer

berfokus pada pengembangan aplikasi web. Mereka bekerja untuk membangun dan memelihara situs web, mulai dari tampilan (frontend) hingga logika dan server (backend).

Software Developer

developer bertanggung jawab untuk merancang, mengembangkan, dan menguji perangkat lunak, baik itu aplikasi desktop, aplikasi web, aplikasi mobile, maupun sistem perangkat lunak lainnya

Mobile App Developer

bertugas untuk mengembangkan aplikasi yang berjalan di platform mobile seperti Android dan iOS

System Analyst

bekerja untuk memahami proses bisnis dan menerjemahkannya ke dalam spesifikasi teknis untuk pengembangan perangkat lunak.
